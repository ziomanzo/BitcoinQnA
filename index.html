Overview

RoninDojo is a dedicated, privacy focused, and user friendly bitcoin node. Our user interfaces, installation scripts, and powerful features are perfect any level bitcoin user. You can count on RoninDojo to provide the tools you need to safely and securley use your Samourai Wallet. By utilizing a dedicated low power device can maintain uptime, small form factor, reliability, low cost, and ease-of-use.

Why RoninDojo?

RoninDojo has a Graphic User Interface currently in beta and a Command Line Interface that is free and open source for those that prefer. This software assists users in quickly deploying a privacy focused bitcoin fullnode. Running a RoninDojo helps you maintain all the security advantages that bitcoin has to offer. Information about your Bitcoin addresses and transactions is verified by RoninDojo and removes the need to trust web-wallets, block explorers, exchanges, and more with information about your funds.
